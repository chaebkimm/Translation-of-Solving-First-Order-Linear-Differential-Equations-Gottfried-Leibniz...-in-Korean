### 서론
1926년 영국의 수학자 에드워드 린제이 인스(1891-1941)는 미적분학에서 (그리고 미분방정식과 일반적인 과학에서) 해를 구하는 테크닉이 어떻게 진화해왔는지를 설명한다. <sup>1</sup>

-----------------------

<p>
미적분학의 초기 역사에는 사실상 미분방정식과 같은 방법으로 문제를 푸는 예가 많았다; 16세기 중반에는 심지어 미분방정식의 가장 간단한 형태의 해로 취급할 수 있는 적분 문제가 실질적인 문제였다고 해도 과언이 아니다. 탄젠트의 역 문제의 특별한 경우 -기울기가 특정 방정식으로 정해진 곡선을 찾는 문제- 는 미적분학이 발명되기 전에도 성공적으로 해결됐다. 
</p>

<p>
하지만 과학의 역사적 가치는 얼마나 많은 특정 현상을 보여주는가에 있지 않고 다양한 사실들을 정리해서 하나의 간단한 코드로 설명할 수 있는 힘에 있다. [인스, 1926]
</p>

<p><sup>1</sup> 인스도 미분방정식에 관한 이야기의 일부이다. 그는 <i>Ince 방정식</i> 을 개발했다 (1923년 경), </p>

```math
(a+acos(2t))y''(t)+(bsin(2t))y'(t)+(\lambda+dcos(2t))y(t) = 0,
```

이는 각각 1868년, 1914년부터 잘 알려진 두개 이상의 다른 방정식들을 일반화한 것이다. $a=b=0$으로 두고 $d=-2q$로 두면, <i>Mathieu 방정식</i> 을 얻을 수 있다 (드럼 모양을 타원형으로 모델링한다), 

```math
y''(t)+(\lambda-2qcos(2t))y(t) = 0,
```

$a=0,b=-4q,d=4q(\nu-1)$로 두면, <i>Whittaker-Hill 방정식 </i>을 얻는다 (달의 안정성과 양자 역학에서 쓰인다)

```math
y''(t)-4q(sin(2t))y'(t)+(\lambda+4q(\nu-1)cos(2t))y(t)=0.
```

Ince 방정식은 또한 <i>일반화된 Ince 방정식 </i>의 특별한 경우이다 ([Moussa, 2014]에서 연구되었다)

```math
(1+\epsilon A(t))y''(t) + \epsilon B(t)y'(t) + (\lambda + \epsilon D(t))y(t)=0.
```

------------------------

이것은 정확히 일계 선형 상미분 방정식 해를 구하는 방법이 진화하는 과정과 같다. 먼저, 특별한 문제가 그 문제를 너머 일반화된 문제에는 적용할 수 없는 "일회성" 방법으로 풀린다. 하지만 이 결과들은 통합된 이론이 개발될 때까지 결합되고 일반화된다. 

과제 1
> [!IMPORTANT]
> 상기된 글에서 인스는 <b>"모든 종류의 미분방정식 중 가장 간단한 미분 방정식의 해"</b>와 <b>"기울기가 특정 방정식에 의해 정해진 곡선을 찾는 문제"</b>를 연결지었다. 두 문장을 연결하라. 미분방정식이
> ```math
> \frac{dy}{dx} = f(x,y),
> ```
> 일 때, <b>곡선,</b>과 <b>기울기,</b>와 <b>특정 방정식</b>은 무엇인가?

과제 2
> [!IMPORTANT]
> 비제차 일계 선형 상미분 방정식이 다음과 같음을 기억하라
> ```math
> p(x)\frac{dy}{dx}+q(x)y = f(x), \tag{1}
> ```
> 최고차항을 1로 만들면,
> ```math
> \frac{dy}{dx}+P(x)y = Q(x), \tag{2}
> ```
> 첫번째 방정식에서 어떻게 두번째 방정식처럼 최고차항을 1로 만들 수 있는지 설명하라. 특히, 왜 우리는 p(x)가 0이 아니라고 가정할 수 있는가? P(x)와 Q(x)를 p(x), q(x), f(x)에 대해서 써보라.

본 프로젝트의 테마는 고트프리드 라이프니츠 (1646-1716)가 했던 것처럼 과제 2에서와 같은 방정식들에 "일회성" 방법을 먼저 적용하는 것이다. 시간이 지나면, 미분방정식의 해들이 변수 치환법, 완전 미분 방정식처럼 더 일반적이고 조화로운 테크닉으로 구해지게 된다. <sup>2</sup>

## 라이프니츠의 확인
1694년 11월 27일, 고트프리드 라이프니츠<sup>3</sup>는 그의 친구인 기욤 드 로피탈<sup>4</sup> (1661-1704) 에게 편지를 쓰는데, 편지에는 1850 종류에 달하는 라이프니츠의 연구 결과가 있었고, 편지의 내용은 훗날 카를 임마누엘 게르하르트 (1816-1899) 정리했다. 편지에는 비제차 일계 선형 상분방정식의 해법도 들어있었다. 

독자는 편지 원본에 등장하는 두개의 표기법을 인지할 필요가 있다 [라이프니츠, 1694]. 먼저, $dy : dx$나 $dp : p$는 단순히 $\frac{dy}{dx}$나 $\frac{dp}{p}$를 의미하며, 오늘날 우리가 콜론으로 비율을 나타내는 것과 동일하다. 둘째로, 기호 $\int \overline{mpdx}$이 제곱근 기호처럼 보일 수 있지만, 사실 적분 기호 $\int$와 윗줄 기호 $\overline{mp\\, dx}$ 두개이다. 윗줄 기호는 괄호와 같이 사용되며 (오늘날 우리가 말하는) 피적분 함수를 뜻한다. 따라서, $\int \overline{mp\\, dx} = \int(mp\\, dx) = \int mp\\, dx$이 된다. 

-------------------

<sup>2</sup> 더 일반화된 방법은 본 "일계 선형 미분 방정식 풀이" 시리즈의 다른 두 프로젝트에서 확인할 수 있고, 두 프로젝트에서는 역사의 발자취를 이어서 따라가며 각각 요한 베르누이 (1667-1748)와 레온하르트 오일러 (1707-1783)의 연구 결과에 대해 알아본다. 본 시리즈의 세 프로젝트는 하나씩 봐도 되고, 이어서 다른 프로젝트에 이어서 봐도 된다. https://digitalcommons.ursinus.edu/triumphs_differ/ 에서 찾아볼 수 있다.

... $m + ny + dy : dx = 0$로 놓으면, m과 n이 변수 x에만 의존하는 유리식 또는 무리식일 때, 일반적으로 $\int \overline{mp\\,dx} + py = 0$ 꼴로 풀 수 있고, 이때 $\int \overline{dp:p} = \int \overline{n\\,dx}$라 가정한다. 해의 양변을 미분하면 $mp\\,dx+y\\,dp+p\\,dy=0$ 이 되는데, $dp=pn//,dx$ 이므로, 대입하면 $mp\\,dx+npy\\,dx+p\\,dy=0$ 또는 원하던대로 $m\\,dx+ny\\,dx+dy=0$이 된다. <sup>5</sup>

------------------

상기된 글을 읽으며, 우리는 라이프니츠가 문제를 거꾸로 풀었다는 것을 알 수 있다. 이는 종종 있는 일로, 미분방정식 또는 관련된 문제에서 해를 <i>찾는</i> 것은 어떤 것이 해인지 <i>확인</i>하는 것보다 훨씬 어렵다. 상기 글에서, 라이프니츠는 두번째 방법을 취했다. 그는 우리가 어떤 함수 p가 식 $\int \overline{dp:p} = \int \overline{n\\,dx}$를 만족하게 정의했을 때 $\int \overline{mp\\,dx} + py = 0$이 $m + ny + dy:dx = 0$의 해라는 것을 확인했다. 

과제 3
> [!IMPORTANT]
>
> <ol type="a">
> <li> 라이프니츠가 어떻게 $\int \overline{mp\\,dx} + py = 0$에서 $mp\\,dx + y\\,dp + p\\,dy = 0$을 얻었는지 자신의 단어로 설명하라.</li>
>
> <li> 라이프니츠가 어떻게 $\int \overline{dp:p} = \int \overline{n\\,dx}$에서 $dp = pn\\,dx$를 얻었는지 자신의 단어로 설명하라.</li>
>
> <li>(c) 라이프니츠가 어떻게 (a)와 (b)를 결합하여 $m\\,dx+ny\\,dx+dy=0$을 얻었는지 자신의 단어로 설명하라.</li>
>
> <li>(d) 라이프니츠는 "원하던대로 식이 된다" 라고 결론지었다. 왜 그 식이 "<b>원하던</b>" 결과인가?</li>
> </ol>

우리는 $\int \overline{mp\\,dx}+py=0$이 미분방정식의 <i>음</i>의 해라는 것을 볼 수 있다. 일반적으로, 해를 구했을 때 <i>음</i>의 해를 얻으면 이를 y에 대해 풀어서 <i>양</i>의 해로 만드는 것이 불가능하다. 다행히, 이번에는 그렇지 않다. 

과제 4
> [!IMPORTANT]
> <ol type="a">
> <li> 라이프니츠의 음의 해를 y에 대해 풀어서 양의 해로 바꾸라.</li>
> </ol>

라이프니츠는 그의 방법이 기존에 알려져 있는 것의 연장선상에 있다는 것을 알았다. 어쩌면 기록된 최초의 미분방정식은 (실질적으로) 일계 선형 미분 방정식이었을지도 모른다! 1638년 프랑스 철학자이자 과학자인 르네 데카르트 (1596-1650)에게 보낸 편지에서, 프랑스 법학자이자 수학자인 플로리몽 드 라몽은 오늘날 우리가 다음과 같이 쓸 수 있는 방정식에 대한 기하학적 해를 질문한다. 

(3)
```math
\frac{dy}{dx} = \frac{\alpha}{(y-x)} \tag{3}
```

이 식은 선형이 아니지만, 과제 5와 같이 선형으로 만들 수 있다 [르누아르, 1979].

m과 n이 x에 대한 유리함수 또는 무리함수인 것은 과제 5에서와 같이 m과 n을 값으로 제한하는 것에 비해 확실히 개선된 것이었다. 하지만 라이프니츠는 그의 개선이 더 통합된 이론을 향해 한발짝 더 나아간 것 뿐이라는 것을 알고 있었다. 미분방정식 테크닉에 이어서 그는 아래와 같이 서술하고 있다.

## 라이프니츠의 직관

특정 미분방정식을 풀 수 있는 비슷한 "트릭"들이 수십년간 문헌에서 성행하였으며, 라이프니츠의 경우와 마찬가지로 이들은 뜬금없이 나타났다. 미분방정식 역사에 대한 연구에서, 딕 쟈르뎅은 그시절 수학자들이 미분방정식을 푸는 방법들을 창조하기 위해 몇시간이고 연습해서 직관을 얻었다고 적었다 [쟈르뎅, 2011]. 결국에는 같은 직관을 통해 그들이 비슷한 트릭들을 정리하여 일반적인 이론을 만들었다. 

--------------------

학생들은 처음에 어떻게 누군가가 이런 관계를 "관찰"하거나 "서술"할 수 있었는지에 대해 당황스러워 한다. 내 최선의 설명은 라이프니츠, 베르누이, 오일러가 오랜 시간을 들여서 이를 포함한 미적분학의 여러 유용한 결과들을 정했다는 것이다. 그들의 노력 덕분에 그들은 그런 관계들에 대한 수학적 직관을 계발했다. 

--------------------

쟈르뎅은 이어서 다음과 같이 결론짓는다, "비슷한 노력을 통해, 우리 학생들은 비슷한 직관을 얻을 수 있다." 어쩌면 당신은
